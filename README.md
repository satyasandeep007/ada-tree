# Adaline Task

## Getting Started

First, run the backend server:

```bash
cd backend
pnpm install
pnpm dev
```

Then, run the frontend server:

```bash

pnpm install

pnpm dev
```

Backend will run on port 9292.

Frontend will run on port 3000.

## Tech Stack

- React
- Next.js
- TailwindCSS
- TypeScript
- Firebase
- @hello-pangea/dnd
- Express
- WebSocket
